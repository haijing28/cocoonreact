# React-Demo

## Reactjs && Node && webpack

![Mou icon](./cocoon.png)

## install & run

clone

```
> $ git clone 
```

|| react && server directory 

```
> $ npm install
```

|| server directory 

```
> $ npm run start
```

|| react directory 

```
> $ npm run dev
```

|| react directory build index

```
> $ npm run build
```

open `http://localhost:8080/' in browersers，

后台服务端口为 `3000`
